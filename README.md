# Rosie Le Xiu — Personal Website

Personal academic website for Rosie Le Xiu, PhD student in Educational Psychology at USC Rossier School of Education.

**Live site:** [rosiexiu.com](https://rosiexiu.com)

## About

This site is built with Jekyll using the [Academic Pages](https://github.com/academicpages/academicpages.github.io) theme, hosted on GitHub Pages.

## Structure

- `_pages/` — Main pages (About, Publications, Talks, Teaching, CV, Portfolio)
- `_publications/` — Publication entries
- `_talks/` — Talk and presentation entries
- `_portfolio/` — Portfolio project entries
- `_teaching/` — Teaching entries
- `_data/` — Navigation and other site data
- `assets/` — CSS, JS, and other assets
- `images/` — Images used on the site
- `files/` — Downloadable files (CV, papers, etc.)

## Running Locally

```bash
bundle install
bundle exec jekyll serve
```

Then visit `http://localhost:4000`.
